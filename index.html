CSS Selector & Specificity Cheatsheet

Selecting proper tags within CSS to target specific HTML elements in your webpage is paramount to both readability of code AND artistic flexibility. This cheatsheet's purpose is to cover the basic tags in regard to their semantic worth and their practical usage or lack thereof.

What are Selectors and how do they contribute to Specificity?

Selectors

In CSS, selectors are the terms used to target the HTML elements on our web pages that we want to style. There are a wide variety of CSS selectors available, allowing for fine-grained precision when selecting elements to style. This applies to most HTML elements including:

  • Headings
  • Articles
  • Lists
  • And many more!

Selectors can also be used in combination to style even more specific elements, or if you want to avoid styling all elements of a particular type.

Specificity

Specificity is the means by which browsers decide which CSS property values are the most relevant to an element and, therefore, will be applied. Specificity is based on the matching rules which are composed of different sorts of CSS selectors. If there are multiple selectors that target a specific element, the most specific one will always take preference.
